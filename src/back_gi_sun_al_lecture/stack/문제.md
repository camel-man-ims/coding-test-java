3. postfix 계산하기

prefix ( 연산자가 앞에옴 )
infix ( 평소의 계산식 )
postfix ( 연산자가 뒤에옴 )

먼저 해야되는걸로 붙여서 적어주면 된다

ex)
1+ 2*3 : infix
+*231 : prefix
123*+ : postfix

ex)
(1+2)*3

*+123 : infix
12+3* : postfix


4. Span 찾기 ( 주식문제 )

ex) 5,3,2,4,7,1 -> [1,1,1,3,5,1]
